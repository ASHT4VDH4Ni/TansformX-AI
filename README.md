# 🏋️ TransformX AI

TransformX AI is an intelligent fitness transformation platform powered by AI agents. It provides personalized workout plans, nutrition guidance, recipe generation, progress tracking, premium health scores, and professional PDF reports.

---

## 🚀 Features

### 🧠 AI Multi-Agent System

TransformX AI uses specialized agents:

* Fitness Coach Agent
* Dietitian Agent
* Motivation Agent
* Progress Analysis Agent
* Injury Prevention Agent
* Recipe Generator Agent
* Reminder Coach Agent
* Progress Revision Agent
* Supervisor Agent

---

## 📋 Personalized 60-Day Transformation Plan

Based on:

* Age
* Gender
* Height
* Weight
* Goal
* Experience Level
* Workout Location
* Diet Preference
* Activity Level
* Sleep Hours
* Existing Injuries

---

## 📊 Body Analysis

Automatically calculates:

* BMI
* BMI Category
* Body Type
* Body Fat Category

---

## 🍽️ Nutrition Planning

Supports:

* Vegetarian diets
* Non-vegetarian diets
* Home workout plans
* Gym workout plans

---

## 🥗 Recipe Generator

Provides:

* Ingredients
* Preparation steps
* Calories
* Protein information

---

## 📈 Progress Tracking

Calculates:

* Progress Score
* Sleep Score
* Hydration Score
* Habit Score
* Transformation Score

---

## 🔥 Premium Intelligence Features

### Dynamic Calorie Adjustment

Automatically recommends calorie changes when progress plateaus.

### Plateau Detection

Identifies slow progress and provides recommendations.

### AI Coach Personality

Instead of robotic responses, TransformX AI provides motivational coaching messages to encourage long-term consistency.

---

## 📄 Professional PDF Reports

Generates downloadable reports containing:

* User Profile
* Weekly Progress Summary
* AI Coach Message
* Calorie Recommendation
* Workout Plan
* Nutrition Plan
* Recipes
* Recovery Strategy

Reports include:

* Dynamic filename
* Generation date and time
* Clean formatting

---

## 📷 Photo Upload System

Users can upload:

* Front photo
* Side photo
* Back photo

These will be used for future visual transformation analysis.
